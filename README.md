# Random Element Generator

A lightweight, interactive web application that demonstrates dynamic **DOM Manipulation** using JavaScript. With every click, a new colorful square is generated at a random position with a unique rotation.

## 🔗 Live Demo

Check out the project in action here:  
**[👉 View Live Demo](https://priyanshjain543.github.io/Random-Element-Generator-TASK-17/)**

---

## 🚀 Features

* **Dynamic Creation:** Uses `document.createElement` to inject new elements into the page.
* **Randomization Logic:**
    * **Colors:** Generates random RGB values for every new square.
    * **Positioning:** Elements appear at random `top` and `left` percentages.
    * **Rotation:** Each element gets a random CSS `rotate` property.
* **Modern UI:** A dark-themed interface with a high-contrast interactive button. 

## 🛠️ Tech Stack

* **HTML5:** Page structure.
* **CSS3:** Flexbox centering, absolute positioning, and button active states.
* **JavaScript (ES6):** Event listeners and math-based randomization.

## 📂 Project Files

| File | Description |
| :--- | :--- |
| `index.html` | The main entry point with the container and button. |
| `style.css` | Handles the dark theme and button animations. |
| `script.js` | Contains the logic for generating the random divs. |

## 📝 How to Use

1.  Clone or download the project files.
2.  Open `index.html` in any web browser.
3.  Click the **"Click Me"** button to populate the screen with random colorful squares.
